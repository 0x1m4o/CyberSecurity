## LKSN IT Software

 - ## [odbg110.zip](https://drive.google.com/file/d/1ZLCiAfdpSryI4Wx17J882vT64JU2ujkj/view?usp=sharing)
#### ❒ OllyDbg is a debugger analysis application. For each module (executable file) it separates code from data, procedures, looks up strings, switch tables, loops, switches, finds functions and decodes arguments, even predicts register values during execution.

 - ## [opensoc-master.zip](https://drive.google.com/file/d/1W4QmKa0c0mzQphcUIf9JhOmW1eL59t4O/view?usp=sharing)
 #### ❒ The OpenSOC is a collaborative open source to providing an extensible and scalable advanced security analytics tool. It has strong foundations in the Apache Hadoop Framework and values collaboration for high-quality community-based open source development.
